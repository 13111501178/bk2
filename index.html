HTML网页介绍

有很多小伙伴不解，HTML到底属于什么语言，是用来干什么的，其实HTML是一种超文本标记语言，用作网页设计，也就是属于前端，包含了html，html5，Javascript，css，css3这些语言都可以对网页进行编辑

Javascript

Javascript，简称也就是JS，它可以使你的网页功能变的有趣味，比如现在屏幕最下方有一个按钮，你点击它会进入我的另一个网页，这就是点击事件，是由JS做成的，大部分网页设计程序员Ctrl C V都是关于Javascript的

css

css是html网页中运用最广泛的东西，比如我这个博客网页，运用了大量的css，css可以使你的网页变得更好看，设置背景图片，字体颜色，大小都是靠css
